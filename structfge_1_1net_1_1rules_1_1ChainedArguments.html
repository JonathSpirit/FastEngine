Detailed Description

template<class TValue>
struct fge::net::rules::ChainedArguments< TValue >

This is a wrapper around a Packet with an optional value.

This structure go through a rules chain and avoid extracting multiple times the required value.

When a rule is invalid, it will invalidate the Packet.

Template Parameters
TValueThe type of the value that will be extracted
